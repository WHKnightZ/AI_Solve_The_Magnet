# AI Solve The Magnet

AI auto solve The Magnet game I've made before

## Content:

Setup:<br />
- Setup MinGW/G++<br />
- Only build main.cpp<br />

Linker:<br />
- Window: -lbgi -lgdi32 -mwindows -m32<br />

Note:<br />

- Arrows: Move
- R: Restart
- Enter: Action
- E: Solve
- Q: Undo
- ESC: Quit
